# mesto-react

Сервис Mesto: интерактивной страницей, куда можно добавлять фотографии, удалять их и ставить лайки.

Технологии, использованные в проекте:
- HTML
- CSS
- Javascript
- Flex
- Grid
- Медиа-запросы для адаптивности сайта
- БЭМ
- Figma
- Popup
- ООП
- WebPack
- Работа с удалённым сервером
- React
- Хуки
